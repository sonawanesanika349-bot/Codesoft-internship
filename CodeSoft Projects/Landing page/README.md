Aurora Jewelry — Landing Page

What this is
- A single-page landing site for a jewelry brand (Aurora Jewels).
- Includes: hero, featured collection, gallery, testimonials, contact form, and a quick-view modal.

Files
- index.html — main page (links Google Fonts and style.css, loads script.js)
- style.css — styling for layout, colors, responsiveness
- script.js — simple interactivity: mobile nav toggle, smooth scroll, quick-view modal, and basic form validation

How to run
1. Open `index.html` in your browser (double-click or use "Open with" -> browser).
2. For development iterate on the files, refresh the browser to see changes.

Notes
- Images are loaded from Unsplash; if you want to use local images, replace the URLs in `index.html` and `style.css`.
- The contact form is client-side only; for production wire it to your backend or a service (e.g., Formspree).
